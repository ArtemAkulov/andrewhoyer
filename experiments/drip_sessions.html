Drip Sessions:

Click and drag within the rectangle below...(or click the demo button)

What's this all about?

On GRL's website, there was a project called "the Drip Sessions" where they would use light to paint graffiti on the sides of entire buildings and structures. I just wanted to create something like that... except with code for the web.

Recoding:

For a while this program wouldn't work with opera and I couldn't figure out why. So instead of tinkering around and wasting time I decided it would be easier to just recode the entire thing. This is about 50 minutes of work sped up so it only takes 3 minutes. Music: Meow by Motohiro Nakashima.
